Bonjour,
Dans ce petit fichier nous allons vous expliquer comment lancer la simulation du projet histoire.
Dans un premier temps je vous informe que vous n'avez pas besoin d'ajoute de librairies.
Une fois le projet ajouter sur eclipse il suffit de lancer la classe ChronosGUI et la simulation démarre.
Vous aurez le choix entre plusieurs années de départ. Une fois votre choix effectué cliquer sur le bouton.(Vous pouvez toujours changer d'avis avant de lancer la simulation)
Pour lancer la simulation vous avez juste à cliquer sur "Commencer". Puis une fois la page finale affichée cliquez sur "Continuer".
Vous pouvez, à tout moment recommencer la simulation à partir de la préhistoire en cliquant sur "Recommencer".
Durant la simulation vous pouvez cliquer sur les zones colorée de la carte pour avoir des informations supplémentaires.
Vous aurez aussi la possibilité de changer le cours de l'Histoire à certains moment.
Pour plus d'informations à propos du fonctionnement n'hésitez pas à consulter le rapport de projet.
